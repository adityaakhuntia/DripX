# DripX - Drip Different

Welcome to **DripX**, a premium, high-fidelity landing page for a luxury footwear brand. Designed with an elegant, modern "light mode" aesthetic, this project demonstrates smooth scroll interactions, glassmorphism UI, and highly engaging scroll-based animations.

![DripX Preview](https://images.unsplash.com/photo-1542291026-7eec264c27ff?w=800&q=90)

## 🌟 Features
- **Elegant Light Aesthetic:** A pristine UI using frosted glass (`backdrop-filter`), extremely soft ambient shadows, and a clean white/gray canvas punctuated by sophisticated accent colors.
- **Scroll & Motion:** Powered by Framer Motion, GSAP, and Lenis smooth scrolling. Experience fluid typography animations, parallax effects on product images, and horizontal-scrolling showcase tracks.
- **Custom Cursor:** Features a custom cursor with an interactive "ambient shadow" trail and dynamic hover states for a unique user experience.
- **High-Fidelity Components:** Custom-built Hero Sections, Infinite Marquee Tickers, Interactive Hover Cards, and a rich Footer.

## 🚀 Tech Stack
- **Framework:** Next.js 15
- **Library:** React 18
- **Styling:** Tailwind CSS (Custom extended design system)
- **Animation:** Framer Motion, GSAP
- **Scroll:** Lenis

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/adityaakhuntia/DripX.git
   cd DripX
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎨 Design System
The UI was meticulously crafted from scratch, shifting from a dark-mode neon theme to this final Elegant Light Mode:
- **Base:** `#ffffff` (Void), `#f8f9fa` (Abyss), `#f1f3f5` (Carbon)
- **Typography:** Deep Charcoals and sleek silvers for high readability and premium feel. Space Grotesk, Bebas Neue, Inter, and JetBrains Mono.
- **Accents:** Cobalt Blue, Rose, and Refined Orange used sparingly for interactive highlights.
